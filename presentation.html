Introduction à Git

Qu'est-ce que Git ?

Git est un système de contrôle de version distribué qui permet de gérer efficacement les modifications du code source.

Commandes Git essentielles

  • git init - Initialiser un dépôt Git
  • git add . - Ajouter les fichiers au suivi
  • git commit -m "Message" - Enregistrer un instantané du projet
  • git status - Vérifier l'état des fichiers
  • git log - Voir l'historique des commits
